# Digital Board DOOH

A Django-based Digital Out-of-Home (DOOH) advertising system that allows real-time ad management, scheduling, and display across multiple digital boards. Designed to provide a flexible platform for clients and administrators to manage and monitor advertisements efficiently.

---

## Features

- Upload and manage ads (images and videos)
- Schedule ads for display on multiple boards
- User roles: Client, Admin, Super Admin
- Real-time ad display updates
- Dashboards for monitoring ads, boards, and users
- Secure login and account management


---

## Technologies Used

- Python 3.11
- Django 4.x
- SQLite (for development)
- HTML / CSS / JavaScript
